# Project Overview

## Project Description

**Spell Arena** is a real-time multiplayer wizard combat game that uses gesture-based inputs to cast spells and compete against other players. The project combines fast-paced dueling with expressive motion-based controls, aiming to create a competitive but approachable experience.

My contribution to the project focuses on improving **player experience**, **accessibility**, and **long-term engagement** through visual theming, progression rewards, and assistive gameplay features.

---

## Problem Domain

While gesture-based multiplayer games can be exciting, they often introduce barriers for new players and struggle with long-term player retention. Common challenges include:

- New players feeling overwhelmed by complex gesture inputs
- Lack of visual identity or cohesive art direction
- Limited incentives to continue playing after initial matches

This project addresses these issues by emphasizing clarity, feedback, and motivation—making the game more welcoming to new players while rewarding mastery and continued play.

# Features and Requirements

## Features & Requirements

### Feature 1: Custom Game Theme and Art Direction

Create a unique visual theme that supports the game’s fantasy wizard combat setting.

**Requirements:**
- Consistent color palette and UI styling
- Themed menus, HUD elements, and effects
- Visual clarity during fast-paced combat
- Theme supports readability and accessibility

**Acceptance Criteria:**
- Players can clearly distinguish UI elements and spell effects
- Visual style remains consistent across menus and gameplay

---

### Feature 2: Match Rewards and Replayability System

Introduce rewards for winning matches to encourage continued engagement and replayability.

**Requirements:**
- Track match outcomes (win/loss)
- Grant rewards upon victory (cosmetic or progression-based)
- Display rewards clearly after each match
- Support future expansion of reward types

**Acceptance Criteria:**
- Players receive visible feedback when earning rewards
- Rewards persist across play sessions
- System encourages repeat matches without affecting competitive balance

---

### Feature 3: Assistive Options for New Players

Provide assistive tools that help new players learn the game without removing skill expression.

**Requirements:**
- Pinned movesets showing available gestures during matches
- Contextual tooltips explaining spells and mechanics
- Guided post-match reports highlighting:
  - Successful spells
  - Missed opportunities
  - Suggested improvements

**Acceptance Criteria:**
- New players can complete matches with reduced confusion
- Assistive elements can be toggled on/off
- Post-match feedback is informative, not overwhelming
